# ACTUAL: weekly screening of the participant environmental and personal data

Author: Tino Schneidewind


### I answer the following questions in this App:

1. Who has been monitored per week/time-period? 
2. Which files are available per person?
3. What were the exposures of each person of the week?


### How to set up this App
1. create a folder in in the place of your choice
2. fork this repository into the folder (we need an additional upper layer to save the data that cannot be uploaded to GitHub)
3. create in the initial folder from (1.) a folder calles "data"
4. now you can open the Rproj in Rstudio, and open the 01_getREDCap.R file
5. run this (you will need the TOKEN for this to work) !dont save the token in this R file!
6. open the app.R file and run it line by line
